# Requisitos Funcionales

## RF-01 Registro de usuarios
El sistema permite que los usuarios puedan registrarse dentro de la plataforma para acceder a las funciones disponibles según su rol.

## RF-02 Inicio de sesión
La aplicación permite iniciar sesión para acceder a las funciones correspondientes de comprador, productor o comité.

## RF-03 Publicación de productos
El productor puede publicar productos agrícolas indicando información como nombre, precio, cantidad e imagen.

## RF-04 Visualización de catálogo
Los usuarios pueden visualizar el catálogo de productos disponibles dentro de la plataforma.

## RF-05 Lista de interés
El comprador puede guardar productos dentro de una lista de interés antes de contactar al productor.

## RF-06 Comunicación entre usuarios
El sistema permite la comunicación entre compradores y productores para coordinar detalles relacionados con productos o pedidos.

## RF-07 Gestión de productos
El productor puede editar, actualizar o eliminar productos publicados dentro del sistema.

## RF-08 Panel del comité
El comité puede revisar información general del sistema y administrar usuarios o contenido.

## Observación
Estos requisitos representan las funciones principales consideradas para el primer release del sistema.
